SeekerOS - Beta Release (v0.1)

Thank you for trying SeekerOS!
This is an experimental firmware for the M5StickC / M5StickC Plus based on ESP32.

--------------------------------------------------
How to Install
--------------------------------------------------
1. Download the SeekerOS-v0.1-beta.bin file from:
   https://yourwebsite.com/SeekerOS-v0.1-beta.bin

2. Open M5Burner (Windows, macOS, Linux available from:
   https://docs.m5stack.com/en/download)

3. Connect your M5StickC device to your computer via USB.

4. In M5Burner:
   - Click "Add Custom Firmware"
   - Enter:
       Name: SeekerOS
       Version: v0.1-beta
       URL: (link to your hosted .bin)
   - Or select the local .bin file directly

5. Flash the firmware to your device.

6. Once done, your M5StickC will reboot into SeekerOS.

--------------------------------------------------
Features in this version
--------------------------------------------------
- Boot splash screen with slogan
- Live time display from onboard RTC
- Battery indicator
- Basic UI screens (Home, Results, Details)

--------------------------------------------------
Notes
--------------------------------------------------
- This is a **beta release**; expect bugs or unfinished features.
- Only tested on M5StickC Plus (ESP32-based).
- Use at your own risk.
